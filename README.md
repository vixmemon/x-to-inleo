# X to InLeo - Chrome Extension

Share your favorite X (Twitter) posts directly to the InLeo blockchain with one click!

## Features

✨ **One-Click Sharing** - Add an InLeo button to every X post for instant sharing to the blockchain

📝 **Smart Content Extraction** - Automatically captures post text, images, videos, and polls

🔐 **Secure Authentication** - Uses Hive Keychain for safe, private key-free authentication

💰 **Earn Rewards** - Get rewarded for sharing quality content to the InLeo community

🎁 **Tip Creators** - Send tips in HIVE or HBD directly to content creators

⚡ **Fast & Lightweight** - Minimal performance impact on your browsing

## How It Works

1. **Install the Extension** - Add X to InLeo from the Chrome Web Store
2. **Connect Your Account** - Link your Hive account using Hive Keychain
3. **Share Posts** - Click the 🦁 InLeo button on any X post
4. **Customize** - Edit content, include source link, and add media
5. **Post to Blockchain** - Share to InLeo and earn rewards

## Requirements

- Hive account (create at https://inleo.io)
- Hive Keychain extension installed
- Chrome/Chromium browser

## Features in Detail

### Share X Posts to InLeo
- Extract text, images, videos, and polls from X posts
- Customize content before posting
- Include source link to original tweet
- Automatic character limit optimization

### Secure Authentication
- No private keys stored locally
- Uses Hive Keychain for signing
- One-time setup, persistent login

### Tip System
- Send tips in HIVE or HBD tokens
- Support content creators directly
- Quick preset amounts (1, 5, 10, 20)
- Custom amount support

### Rate Limiting
- 30-second cooldown between posts
- Prevents spam and accidental duplicates
- Minimum 10 character post requirement

## Getting Started

1. Install Hive Keychain: https://chromewebstore.google.com/detail/hive-keychain/jcacnejopjdphbnjgfaaobbfafkihpep
2. Create a Hive account: https://inleo.io/signup?referral=leoquoter
3. Import your account in Hive Keychain
4. Install X to InLeo extension
5. Click the extension icon and connect your Hive username
6. Start sharing!
